Card Memory Game

A memory card game built using Java FX.

Click Here to See Code!

The user is presented with a board of 8 pairs of cards. The user is tasked with getting 3 matches in a row in order to win the game. The user can quit using the button on the bottom left side of the window at any time. The user clicks two different cards, and the interface tells the user whether they made a match or not. The interface also keeps track of how many consecutive matches the user has made. See images below.
